    The redirect controller has the sole purpose of turning a JSON or JS request into an HTML request by sending a response that redirects to a page

    # File app/controllers/redirect_controller.rb, line 5
def go
  flash.keep # Hold flash for the redirect
  session[:goto] = params[:to]
  respond_to { |format|
    # Decode the URL embedded in the :to parameter, removing enclosing quotes
    format.json {
      render json: { redirect: (params[:to].sub /^"?([^"]*)"?/, '\1') }  # Strip enclosing quotes, if any
    }
  }
end
